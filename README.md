# Sylvie Gagnon Industrial Line Card

A Next.js landing page for presenting represented industrial manufacturers, product categories, and contact information.

## Online preview (Vercel)

Use the button below to deploy this repository instantly:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/<your-org-or-user>/Sylvie-Gagnon)

> Replace `<your-org-or-user>` with your actual GitHub owner before sharing this link.

### Manual deploy steps

1. Push this repository to GitHub.
2. Log in to [Vercel](https://vercel.com).
3. Click **Add New... → Project**.
4. Import this repository.
5. Keep defaults (Vercel auto-detects Next.js).
6. Click **Deploy**.

After deployment, Vercel gives you a public URL like:

- `https://sylvie-gagnon-linecard.vercel.app`

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Scripts

- `npm run dev` — run development server
- `npm run build` — create production build
- `npm run start` — start production server
- `npm run lint` — run Next.js lint checks
